# To-Do List Application

A simple command-line based To-Do List application written in Python.  
This project allows users to manage their daily tasks efficiently by adding, viewing, completing, and deleting tasks.

## Project Description

This application is designed to demonstrate fundamental programming concepts in Python such as:

- Functions
- Loops
- Conditional statements
- Lists (Data Structures)
- User input handling
- File operations (if you used file saving)

The system runs in a terminal environment and provides an interactive menu for task management.

## Features

- Add new tasks
- View all tasks
- Mark tasks as completed
- Delete tasks
- Save tasks (if file system used)
- Continuous menu loop until user exits

## How It Works

The program operates using a loop-based menu system:

1. The user selects an option from the main menu.
2. Based on the input, a specific function is executed.
3. Tasks are stored in a Python list.
4. Each task may contain:
   - Task description
   - Completion status (True/False)

If file handling is implemented:
- Tasks are written to a file.
- On startup, tasks are loaded from the file.
